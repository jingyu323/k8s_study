### 1.Vue keep-alive后不能触发destroyed

解决方法有2种

1.直接调用deactivated

 deactivated() {
    clearInterval(this.timer)
  },
2.配合导航守卫，手动调用生命周期销毁函数

beforeRouteLeave (to, from, next) {
    this.destroy();
    next();
} 

回来之后activated 不触发 ，是activated  而不是actived
